# perl-nes

A NES Emulator written in Perl.  The goal of this project is to write a fully functioning NES emulator in Perl.

# Perl requirements

perl-nes requires SDL.  To install SDL use
	``` perl -MCPAN -e 'install("Alien::SDL");' ```
